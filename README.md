# Airbnb-clone

Live at Netlify: https://delicate-faun-e5630b.netlify.app/

(CTRL+click or CMD+click to open in new tab)

## Steps to Execute locally
- git clone https://github.com/prasantmahato/airbnb-clone.git
- cd airbnb-clone
- npm install 
- npm run dev
